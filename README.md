# Personal Tech Learning Agent 🚀

An AI-Powered Personal Learning Agent built using Java, Spring Boot, and Gradle to automate, track, and optimize technical learning workflows.

---

## 🌟 Features
- **AI-Driven Roadmap Generation:** Dynamically creates customized learning paths based on technology stacks.
- **Smart Knowledge Tracking:** Tracks learning progress and revision schedules.
- **Automated Summarization:** Extracts key insights and summaries from technical documentation and articles.
- **Interactive Quizzing:** Generates contextual flashcards and MCQs to test retention.

---

## 🛠️ Technology Stack
- **Backend:** Java 17 / 21, Spring Boot 3.x
- **Build Tool:** Gradle
- **Database:** PostgreSQL / MongoDB (or your preferred DB)
- **AI Integration:** Spring AI / OpenAI API / Gemini API

---
